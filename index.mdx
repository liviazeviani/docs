TaskFlow is a task management API for creating projects and organizing the tasks within them.

## Core concepts

TaskFlow is built around two resources:

<CardGroup cols={2}>
  <Card title="Projects" icon="folder">
    Containers that group related tasks together. Every task belongs to exactly one project.
  </Card>

  <Card title="Tasks" icon="list-check">
    Individual units of work inside a project. Tasks can be created, updated, listed, and deleted.
  </Card>
</CardGroup>

In practice, you'll typically create a project first, then create and manage tasks within it.

## Authentication

All requests to the API require a Bearer token in the `Authorization` header:

```bash
Authorization: Bearer YOUR_API_KEY
```

See the [Authentication](/authentication) page for details on how to obtain and use your API key.

## Base URL

All API requests are made against:

```http
https://test-writechoice.onrender.com/
```

<Note>
  The API may take up to 60 seconds to respond on the first request if it has been idle. Subsequent requests will be fast.
</Note>
