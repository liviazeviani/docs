TaskFlow is a simple task management API. It provides endpoints to create projects, organize tasks within those projects, retrieve task lists, update task information, and delete tasks.

<Tip>
  The fastest way to learn the API is by following the Quickstart guide, which walks through creating a project and managing tasks step by step. [Quickstart](/quickstart) page.
</Tip>
